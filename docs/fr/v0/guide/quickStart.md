# Démarrage rapide

## Installation des dépendances
::: code-group
```bash [npm]
npm install @duplojs/json-web-token@0 @duplojs/utils@1
```
```bash [yarn]
yarn add @duplojs/json-web-token@0 @duplojs/utils@1
```
```bash [pnpm]
pnpm add @duplojs/json-web-token@0 @duplojs/utils@1
```
```bash [bun]
bun add @duplojs/json-web-token@0 @duplojs/utils@1
```
```bash [deno]
deno add npm:@duplojs/json-web-token@0 npm:@duplojs/utils@1
```
:::
